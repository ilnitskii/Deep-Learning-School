f# Deep Learning School Projects

Collection of machine learning and deep learning projects completed during the **Deep Learning School (MIPT)** program.

The repository covers a wide range of topics including:

* classical machine learning
* computer vision
* natural language processing
* generative models
* speech processing
* modern deep learning architectures

The goal of this repository is to demonstrate practical implementation of deep learning methods across multiple domains.

---

# Topics Covered

## Classical Machine Learning

Basic ML algorithms and model evaluation.

Projects:

* Linear models
* Binary classifiers
* Feature engineering and evaluation metrics

---

## Computer Vision

Deep learning models for visual data.

Projects:

* CNN image classification
* Semantic segmentation of images
* Object detection

---

## Generative Models

Implementation of generative neural networks.

Projects:

* Variational Autoencoders (VAE)
* Generative Adversarial Networks (GAN)

---

## Natural Language Processing

Modern NLP techniques and architectures.

Projects:

* Text classification
* Language modeling
* Transformer attention mechanisms
* Retrieval-Augmented Generation (RAG)

---

## Speech and Audio Processing

Deep learning models for audio tasks.

Projects:

* Audio classification
* Automatic Speech Recognition (ASR)
* Text-to-Speech (TTS)
* Self-supervised audio models


---

# Technologies

Main tools used in the projects:

* Python
* PyTorch

* Pandas
* Matplotlib
* Scikit-learn
* Librosa
* Torchaudio
